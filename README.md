# Keep


Info
--------------------------------
Project Name: Keep
Node version: v16.13.0
Npm version: 8.1.0


Instructions
--------------------------------

In order to run the project, please clone the repo and then run the following commands:

go to `server` folder:
- run `npm install`
- run `node index.js`

go to `client` folder:
- run `npm install`
- go to `my-app` sub directory
- run `npm start`
